Calculadora para Bordadeiras

Oie, bordadeira! Essa calculadora foi feito para que você possa precificar o seu trabalho de forma justa. Para que funcione, é muito importante que você preencha todos os espaços, ok? Caso seja algo que não faça parte do seu preço, digite apenas um 0.
